# Save your pasted images here

The redesign references these exact files. Save each pasted image with the
exact name + path below (PNG). Once saved, the site shows them immediately —
no code changes needed.

## Hero (full-bleed background scene)
- `public/images/hero-scene.png`   ← the big cinematic facility + sugarcane + glowing product streams

## Products (card images)
- `public/products/fuel-pellets.png`    ← brown pellets on soil
- `public/products/biochar.png`         ← black charcoal heap
- `public/products/seedlings.png`       ← green sprouts in soil  (Slow-Release Fertilizer card)
- `public/products/water-droplet.png`   ← teal droplet on dark soil  (Filter Media card)
- `public/products/pyrolysis-oil.png`   ← amber oil in glass bottle  (Pyrolysis Oil card)

The 6th product card (Carbon / Graphene) keeps the existing vector
`public/products/carbon-graphene.svg` — replace it too if you have a photo.

Tip: in VS Code you can drag an image file directly into the `public/images`
or `public/products` folder in the Explorer, then rename it to match.
